# BkChatAi Final

এই প্রজেক্টটি Netlify-ready।

## যা আছে
- ChatGPT-style UI
- Developer button
- Kamrul Hasan biography
- “তোমাকে কে বানিয়েছে” প্রশ্নের জন্য custom reply
- OPENAI_API_KEY দিলে real AI response

## Folder Structure
- index.html
- netlify.toml
- netlify/functions/chat.js

## Deploy
1. ZIP extract করো
2. GitHub-এ upload করো
3. Netlify-তে import করো
4. Environment Variable এ `OPENAI_API_KEY` যোগ করো
5. Deploy করো

## Environment Variable
Name: OPENAI_API_KEY
Value: তোমার OpenAI API key
